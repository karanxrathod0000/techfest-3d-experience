## 📖 Overview

This is an **immersive, interactive 3D website** built for **Techfest IIT Bombay** — Asia's largest Science & Technology festival. The website showcases the **30th Edition** of Techfest, featuring:

- 🌟 **Immersive 3D Scene** powered by Three.js
- 🎨 **Cyberpunk Aesthetic** with neon cyan and purple glow effects
- 🎮 **Interactive Elements** with mouse and scroll-based animations
- 📱 **Fully Responsive** design for all devices
- 🔥 **GSAP Scroll Animations** for a cinematic experience
- ✨ **UnrealBloomPass** for stunning glow and bloom effects

---

## 🎯 Features

### 🎨 Visual & Design
- **Cyberpunk Theme** with dark background and neon accents (#00F0FF, #8B5CF6)
- **Glass-morphism UI** elements with backdrop blur
- **Animated Typography** with Google Fonts (Orbitron & Inter)
- **Dynamic Lighting** with moving point lights
- **Custom Scrollbar** for enhanced UX

### 🎮 3D & Interactivity
- **Rotating Geometric Centerpiece** with emissive glow
- **1500+ Particle System** forming dynamic starfields
- **Mouse Parallax Effect** — scene reacts to cursor movement
- **Scroll-Triggered Animations** — each section transforms the 3D scene
- **Interactive 3D Objects** representing key events (Robowars, Technoholix, etc.)
- **Animated Counters** for festival statistics

### 📱 Responsive & Performance
- **Mobile-First Design** with touch support
- **Adaptive Particle Count** based on device capability
- **Optimized Rendering** for smooth 60fps experience
- **Single HTML File** — everything self-contained
